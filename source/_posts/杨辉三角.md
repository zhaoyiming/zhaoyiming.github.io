今天参加了一个小笔试，提到杨辉三角居然又忘了，做了无数次...
看来没真的理解。
主要卡在“这中间的空格怎么实现啊！！”
发现这个空格无需关注。
上代码
```
#include <stdio.h>
int main(void)
{
	int a[6][6];
	int i,x,y;
	for(x=0;x<6;x++) {
		for(i=0;i<6-x;i++){
			printf(" ");
		}
		for(y=0;y<=x;y++){
			if(y==0||x==y)
				a[x][y]=1;
			else
			a[x][y]=a[x-1][y]+a[x-1][y-1];
			printf("%d ",a[x][y]);
		}
		printf("\n");		
	}
	return 0;
} 

```
